# Sergio's Projects

Welcome to the landing page for my web development works. This repository serves as a central hub for various projects and applications I've built.

## ✨ Features

- **Modern Glassmorphism Design**: A clean and sleek user interface using modern CSS techniques.
- **Interactive Elements**: Hover effects with glowing gradients and smooth transitions.
- **Animated Background**: Dynamic floating orbs with CSS animations.
- **Responsive Layout**: Fully responsive design that works across different screen sizes.
- **Typography**: Uses the "Outfit" font family from Google Fonts for a professional look.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables), Flexbox, Grid, and Keyframe animations.
- **GitHub Actions**: Automated deployment to GitHub Pages.

## 🚀 Projects Included

- **NeoJackpot**: A slot machine web application.
- **Web App**: Main portal for advanced web applications.

## 📦 Deployment

This site is automatically deployed to GitHub Pages via the workflow defined in `.github/workflows/static.yml`. Every push to the `main` branch triggers a new deployment.
